# Tugas Pemrosesan Citra Digital - Filtering Noise

## Deskripsi
Program ini digunakan untuk menghilangkan noise pada gambar menggunakan berbagai filter berbasis domain frekuensi.

## Metode Filter
- Low Pass Filter  
- High Pass Filter  
- Notch Filter  
- Median Filter  
- Gaussian Filter  

## Tools yang Digunakan
- Google Colab  
- Python (OpenCV, NumPy, Matplotlib)

## Hasil Analisis
- **Median Filter** menghasilkan gambar dengan noise paling sedikit.  
- **High Pass Filter** menghasilkan noise paling banyak.

## Cara Menjalankan Program
1. Buka file `filtering_frekuensi.ipynb` di Google Colab  
2. Upload gambar `image_w_noise.png`  
3. Jalankan semua cell dari atas ke bawah  
4. Amati hasil dari setiap filter
